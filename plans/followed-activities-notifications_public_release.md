# 🔔 Nouveauté : Suivez vos activités préférées et recevez les notifications !

Chers membres,

Nous sommes ravis de vous présenter une nouvelle fonctionnalité qui va vous aider à ne plus manquer les collectives de vos activités favorites !

## Qu'est-ce qui change ?

### 🎯 Suivi automatique des activités

Désormais, lorsque vous vous inscrivez à une collective, vous suivez automatiquement l'activité associée (escalade, randonnée, ski, etc.). Cela vous permet de :

- **Recevoir des notifications** par email dès qu'une nouvelle collective est publiée pour cette activité
- **Suivre plusieurs activités** et rester informé de toutes les sorties qui vous intéressent
- **Gérer vos préférences** depuis votre profil

### 📧 Comment ça marche ?

1. **Inscription = Abonnement** : Quand vous vous inscrivez à une collective, vous suivez automatiquement ses activités
2. **Notification par email** : Lorsqu'un encadrant publie une nouvelle collective, les abonnés reçoivent un email
3. **Un seul email par collective** : Même si une collective concerne plusieurs activités que vous suivez, vous ne recevez qu'un seul email
4. **Désabonnement facile** : Chaque email contient un lien pour vous désabonner d'une activité si vous le souhaitez

## Comment gérer mes activités suivies ?

Rendez-vous dans votre **profil** → **"Mes activités suivies"** pour :
- Voir la liste des activités que vous suivez actuellement
- Vous abonner manuellement à de nouvelles activités
- Vous désabonner des activités qui ne vous intéressent plus

**Important** : Si vous vous désabonnez explicitement d'une activité, vous ne serez plus réabonné automatiquement même si vous participez à une collective de cette activité. Il faudra vous réabonner manuellement depuis votre profil.

## Pour les encadrants

Un nouveau bouton **"Notifier les abonnés"** est disponible sur vos collectives publiées. Ce bouton permet d'envoyer une notification par email à tous les membres qui suivent les activités de votre collective.

**À noter** :
- Le bouton n'apparaît que sur les collectives au statut "Confirmé" (publiées)
- Chaque membre ne reçoit qu'une seule notification par collective
- Les membres peuvent se désabonner à tout moment

## Questions fréquentes

**Q : Vais-je recevoir beaucoup d'emails ?**
R : Vous ne recevrez un email que lorsqu'une nouvelle collective est publiée pour une activité que vous suivez, et seulement si l'encadrant choisit d'utiliser la fonction de notification.

**Q : Comment m'assurer de ne pas manquer les collectives de mon activité favorite ?**
R : Vérifiez dans votre profil que vous suivez bien cette activité. Si ce n'est pas le cas, cliquez sur "Suivre".

**Q : Je me suis désabonné par erreur, comment faire ?**
R : Rendez-vous dans votre profil → "Mes activités suivies" et cliquez sur "Suivre" pour l'activité concernée.

**Q : Est-ce que mes données sont protégées ?**
R : Oui, seuls les encadrants peuvent voir qui suit une activité, et les emails de notification contiennent un lien de désabonnement sécurisé personnel.
