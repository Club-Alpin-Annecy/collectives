Documentation for Collectives!
=======================================

Welcome to the documentation for the ``collectives`` website. It has been made
to help you understand, and thus, participate to the enhancement of this
project.

This is the collectives site to plan events within a mountain sport club.
It is powered by Flask. If any question, please contact the `digital team of
CAF Annecy <mailto:digital@cafannecy.fr>`_

Test and demo website can be found there: `test.collectives.cafannecy.fr
<https://test.collectives.cafannecy.fr>`_

Code, participation, and issue reporting can be done on github: `https://github.com/Club-Alpin-Annecy/collectives
<https://github.com/Club-Alpin-Annecy/collectives>`_
